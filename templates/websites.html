2.  Authentication Web App :

The web app is a kind of prototype I built while learning the Django framework. Although it does not have a good frontend, the backend is very secure. No user can access any content without login credentials.So the user needs to SignUp to get login credentials for future reference to content.A person can SignUp with unique username and email address only. If someone forgets his/her password, he/she can reset the password

by getting a link on the registered email.

Further the web app uses CSRF token to prevent Cross Site request forgery attack. CSRF tokens can prevent CSRF attacks by making it impossible for an attacker to construct a fully valid HTTP request suitable for feeding to a victim user.Further the web app has an SSL certificate to prevent HTTP requests to be intercepted.

Have a look at it by clicking the button below:

Authentication Web App

I was trying to put my website in an iframe tag but was unable to do that because Django contains clickjacking protection in the form of the X-Frame-Options middleware  which, in a supporting browser, can prevent a site from being rendered inside a frame which definitely makes the webapps more secure.
